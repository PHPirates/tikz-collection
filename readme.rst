How to include TikZ pictures
----------------------------

Include the package ``tikzscale``, then put in a file ``stuff.tikz``

.. code:: tex

    \begin{tikzpicture}
        ...
    \end{tikzpicture}
    
and in your main LaTeX file:

.. code:: tex

    \begin{figure}
        \input{stuff.tikz}
        \caption{Description}
        \label{fig:stuff}
    \end{figure}
